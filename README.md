## Introduction

로봇 위치 제어를 통해 케이블 끝단에 대한 위치 제어를 위하여, 로봇과 케이블을 포함하는 시스템에 대한 동적 모델을 심층 신경망 (Deep Neural Network)을 이용하여 학습
